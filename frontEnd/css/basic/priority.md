# 样式优先级
#### 所谓优先级，指的是哪条样式规则会最后作用于指定的元素，遵循4条规则：
    1. !important;高于一切;
    2. 指定的越具体优先级越高；
    3. 相同类型、级别，是后者覆盖前者。
    4. 内嵌样式>内联（内部）样式>外联样式

* 优先级顺序：

!important>内嵌>组合选择器>id选择器>类名选择器>标签选择器>通配符*>浏览器预定义样式>继承的样式


**!important写法上的陷阱：**
* 一定是在分号前！下面写法是错误的，不会起到作用!
```
    color:green;!important;
```

* 正确写法：
```
    color:green!important;
```
