# css样式几种引入方式

### 1.内嵌样式
* 在标签上写
```
<span style="background-color:#f00;">haha</span>
```

### 2.内联（行内）样式
```
<head>
  <style>
         #id2{
            background-color:#0f0;
         }
   </style>
</head>
```



### 3.外部样式表
```
<head>
<link rel="stylesheet" href="css/1.css" type="text/css"/>
</head>
```
