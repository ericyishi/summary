# 发布订阅
### 概念
* Redis 发布订阅(pub/sub)是一种消息通信模式：发送者(pub)发送消息，订阅者(sub)接收消息
* Redis 客户端可以订阅任意数量的频道
* 发送者发到频道的消息，将会被推送到所有订阅此频道的订阅者
* 订阅者不需要主动去获取消息，只需要订阅频道，这个频道的内容就会被推送过来

### 命令
##### 订阅
```html
 SUBSCRIBE 频道名称 [频道名称 ...]
```
##### 取消订阅
```html
 UNSUBSCRIBE 频道名称 [频道名称 ...]
```

##### 发布
```html
 PUBLISH 频道 消息
```
