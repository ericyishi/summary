# 好用的工具
### Linux相关的
* Filezilla
  * 默认使用的协议是ftp，使用sftp需要补全，同时修改端口为22
* nppFPF小插件
  * 是notepad的一个插件，可以方便的对Linux里的文件进行文档编辑。避免直接在linux环境上使用vi命令。
    ```
	  相关说明与下载的文章：https://www.cnblogs.com/0813lichenyu/p/8319124.html
	```
* LICEcap录屏
  * 用于测试复现步骤的录屏，生成gif图
* Snipaste
  * 可以截屏粘贴在桌面上，用于临时对比
* Everything
  * 全盘快速搜索
